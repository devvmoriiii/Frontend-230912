CSS Box Model - margin

margin : 두 박스 모델(요소) 사이의 여백으로, 요소와 주변 요소 간의 간격을 만들 때 사용한다.
상, 우, 하, 좌의 순서로 각각 위치의 여백을 따로 지정하거나 4가지 방향에 동일하게 지정 가능하다.
특정 방향만 지정할 경우 margin-방향 속성을 별도로 사용한다. (margin-top, -bottom, -left, -right)

    중앙 정렬 방법
  1. 배치할 요소의 너비값을 지정하고 margin-right, -left 속성 값을 auto 로 지정한다.
  2. 웹 브라우저 화면의 너비에서 요소 너비값을 뺀 나머지 영역을 좌우 마진으로 자동 계산된다.
    속성 값
  • 크기 : 절대 값과 상대 값으로 적용 가능
  • 백분율 : 부모 요소를 기준으로 상대적인 단위
  • auto : 브라우저가 마진을 자동으로 계산하여, 중앙 정렬 등에 활용 가능 (텍스트 스타일 text-align 과 유사한 개념)
    display 속성에서 지정한 값에 맞게 적절한 값을 자동으로 지정한다.
    값 개수
  • 1개의 값 : 4방향 모두 동일
  • 2개의 값 : 상하 동일 & 좌우 동일
  • 3개의 값 : 상 & 하 & 좌우 동일
  • 4개의 값 : 상 하 좌 우 각각 적용
