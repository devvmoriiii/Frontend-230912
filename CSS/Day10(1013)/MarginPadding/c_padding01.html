CSS Box Model - Padding

padding : 콘텐츠 영역과 border 사이의 여백을 설정
상, 우, 하, 좌의 순서로 각각 위치의 여백을 따로 지정하거나 4가지 방향에 동일하게 지정 가능하다. (margin 과 동일)
특정 방향만 지정할 경우 padding-방향 속성을 별도로 사용한다. (padding-top, -bottom, -left, -right)

    속성 값
  • 크기 : 절대 값과 상대 값으로 적용 가능
  • 백분율 : 부모 요소를 기준으로 상대적인 단위
    값 개수
  • 1개의 값 : 4방향 모두 동일
  • 2개의 값 : 상하 동일 & 좌우 동일
  • 3개의 값 : 상 & 하 & 좌우 동일
  • 4개의 값 : 상 하 좌 우 각각 적용
