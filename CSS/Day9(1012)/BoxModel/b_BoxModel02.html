박스 모델의 기본 구성

웹 문서의 '블록 레벨 요소' 는 모두 박스 형태이다.
콘텐츠 영역 < padding 영역 < border 영역 < margin 영역 의 순으로,

  • 콘텐츠 영역 : 박스 모델의 중심 부분으로, 텍스트 | 이미지 | 다른 미디어 등의 실제 컨텐츠를 포함한다.
    width, height 속성으로 영역의 크기를 정의 가능하다.

  • 콘텐츠와 border 사이의 공간으로, 내부 여백을 생성한다.
    padding-top, -bottom, -left, -right 등 padding 속성을 사용하여 각가의 방향에 대한 패딩 크기를 설정 가능하다.

  • border 영역 : 콘텐츠와 padding 을 둘러싼 외부 요소와의 경계를 정의한다.
    border 속성을 사용하여 테두리의 두께, 스타일, 색상, 모서리의 둥근 정도를 조절 가능하다.

  • margin 영역 : 박스 외부의 공간으로, 다른 요소들과의 간격을 생성한다.
    margin-top, -bottom, -left, -right 등 margin 등 속성을 사용하여 각각의 방향에 대한 마진 크기를 설정 가능하다.

박스 모델 속성

  • width, height 속성 : 콘텐츠 영역의 너비와 높이를 지정하는 속성
    1. 크기 (px, em 등)
    2. 백분율
    3. auto : 콘텐츠의 양에 따라 자동으로 결정 (기본값)

  • box-sizing 속성 : 웹 문서에서 여러 가지 요소 배치 시 실제 박스 모델이 차지하는 크기를 지정하는 속성
    웹 문서 레이아웃을 만들 때는 요소의 크기를 쉽게 계산하기 위해 속성 값을 border-box 로 지정하는 것을 권장

    1. content-box : 요소의 width, height 속성이 콘텐츠 영역에만 적용되며, padding 과 border 는 별도로 추가된다. (기본값)
    2. border-box : 요소의 width, height 속성에 더불어 padding, border 까지 포함하여 계산된다.

  • box-shadow 속성 : 박스 모델에 그림자 효과를 주는 속성으로, 이미지 | div 요소와 같은 전체 영역에 지정하여 삽입한다.
    {수평 수직 흐림 번짐 색상} 순서로 공백으로 구분하여 작성한다.

    1. 수평 거리 (필수) : 그림자가 가로로 얼마나 떨어져 있는지 표시 (양수: 오른쪽 | 음수: 왼쪽)
    2. 수직 거리 (필수) : 그림자가 세로로 얼마나 떨어져 있는지 표시 (양수: 아래쪽 | 음수: 위쪽)
    3. 흐림 정도 : 그림자의 흐림 정도를 지정하며, 수로 값이 커질수록 부드러운 그림자가 표시 (음수 X) (기본값 : 0)
    4. 번짐 정도 : 그림자의 범짐 정도를 지정하며, 값이 커질수록 모든 방향으로 퍼진다 (양수: 모든 방향으로 퍼짐 | 음수 : 모든 방향으로 축소) (기본값 : 0)
    5. 색상 : 한 가지만 지정하거나, 공백으로 구분하여 여러 색상을 지정 (기본값 : #000)
    6. inset : 해당 키워드 사용 시 안쪽으로 그림자가 생성

위의 두 박스는 동일하게 {너비 200px, 높이 200px, padding 20px, border 10px} 로 지정하였으며,
Content Box 는 너비와 높이 값에 별도로 + padding + border 가 계산되므로 200 + (20 * 2) + (10 * 2) = 260 X 260의 크기를 가진다.
Border Box 는 너비와 높이 값에 padding, border 를 포함시키므로 200 - (20 * 2) - (10 * 2) = 140 X 140 의 크기를 가진다.
