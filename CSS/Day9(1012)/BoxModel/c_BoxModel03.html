박스 모델의 방향

: 테두리, 마진, 패딩을 지정할 때 상하좌우 4개의 방향을 똑같이 지정하거나, 모두 다르게 지정 가능하다.
순서는 상단 (top) > 오른쪽 (right) > 하단 (bottom) > 왼쪽 (left) 의 시계 방향이다.

border 속성

: 테두리를 지정하는 속성으로, style & width & color 속성을 같이 사용 가능하다.

    style 속성 값
  • none : 테두리 없음
  • solid : 실선
  • dashed : 대시선
  • dotted : 점선
  • double : 두 줄 테두리
  • groove : 테두리를 창에 조각한 것처럼 표시 (입체적 표시)
    width 속성 값
  • 예약어 : 예약어를 사용하여 지정 (thin, medium, thick)
  • 크기 : 크기 단위를 이용하여 지정 (px, em 등)

border-radius 속성

: 둥근 테두리를 만드는 속성으로, 박스 모델의 꼭짓점 부분에 원이 있다고 가정하여 둥글게 처리한다.
속성 값은 크기 또는 백분율을 사용한다.

  • 타원 처리 : 가로 반지름과 세로 반지름을 / (슬래쉬) 로 구분하여 작성 (ex. border-radius: 50px / 30px)
  • 특정 꼭짓점만 처리 : border-위치-radius 속성을 사용 (ex. border-bottom-left-radius: 30px)
  • 특정 꼭짓점만 타원 처리 : border-위치-radius 속성을 사용하되, 가로 반지름과 세로 반지름을 공백으로 구분하여 작성 (ex. border-bottom-right-radius: 50% 30%)
