목록 스타일

  • caption-side 속성 : 표 제목의 위치를 정해주는 속성

    1. top : 표 위에 위치 (기본값)
    2. bottom : 표 아래에 위치

  • border 속성 : 요소의 테두리를 스타일링하는 속성
    아래 3가지 속성을 'border: 너비 스타일 색상' 형태로 공백으로 구분하여 한 줄에 작성한다.

    1. border-style : 테두리 스타일을 지정
      1. none : 테두리 없음
      2. solid : 실선
      3. dashed : 대시선
      4. dotted : 점선
      5. double : 두 줄 테두리
    2. border-width : 테두리 너비를 지정 (px, em)
    3. border-color : 테두리 색상을 지정

  • border-spacing 속성 : 셀과 셀 사이의 여백을 지정하는 속성으로, 표와 셀에 따로 테두리 지정 시 셀과 셀 사이에 여백 생성
    수평거리와 수직거리의 값을 차례대로 공백으로 구분하여 작성하며, 두 값이 같다면 1개만 작성 가능
    {border-collapse : collapse} 속성을 가질 경우 적용되지 않는다.

    1. 수평거리
    2. 수직거리

  • border-collapse 속성 : 표와 셀 테두리를 합쳐주는 속성
    table, td 태그에서 border 속성 사용 시 여백이 생성되는데, 해당 여백을 합쳐준다.

    1. separate : 표와 셀의 테두리를 따로 표시
    2. collapse : 표와 셀의 테두리를 합쳐서 하나로 표시

표준 테이블 속성

table, td, th {
border: 1px solid #000;
border-collapse: collapse;
}
