목록 스타일

  • list-style-type 속성 : 불릿 모양과 번호 스타일을 지정하는 속성

      ul 태그
    1. disc : ● (채운 원 모양의 불릿)
    2. circle : ◦ (빈 원 모양의 불릿)
    3. square : ■ (채운 사각형 모양의 불릿)
    4. none : 불릿 없음

      ol 태그
    1. decimal : 10진수 (1 ~ 9)
    2. decimal-leading-zero : 숫자 앞에 '0' 이 붙는 10진수 (01 ~ 09)
    3. lower-roman : 로마 숫자 소문자
    4. upper-roman : 로마 숫자 대문자
    5. lower-alpha | lower-latin : 알파벳 소문자
    6. upper-alpha | upper-latin : 알파벳 대문자
    7. none : 번호 매김 없음

  • list-style-image 속성 : 불릿 대신 이미지를 사용하는 속성
    불릿 이미지는 불릿 크기만큼 줄여서 사용하는 것을 권장

    1. url : 'url(이미지 파일 경로)' 형태로 작성
    2. none : 불릿 이미지 없음

  • list-style-position 속성 : 불릿이나 번호의 위치를 들여쓰기하는 속성
    텍스트 문단 사이에 있는 목록을 더 쉽게 구분한다.

    1. outside : 들여쓰기처리되지 않음 (기본값)
    2. inside : 불릿 | 번호가 실제 내용이 시작되는 위치보다 더 들여쓰기 처리

  • list-style 속성 : 위의 속성들을 한 번에 적용할 수 있는 속성
    중복되는 속성 값이 없어 순서에 상관없이 공백으로 나열하여 작성 가능
