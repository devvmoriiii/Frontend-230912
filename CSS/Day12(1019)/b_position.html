static : 특별한 위치 속성 없이 일반적인 문서 흐름에 따라 요소가 배치 (기본 값)

relative (상대적인) : static + 위치 값을 지정할 수 있음 (좌우상하 속성 사용 가능)

absolute (절대적인) : relative 값을 사용한 상위 요소를 기준으로 위치를 지정하여 배치 / 가장 가가운 위치가 지정된 부모 요소에 대해 상대적으로 배치

fixed (고정적인) : 브라우저 창을 기준으로 위치를 지정하여 배치 / 스크롤을 해도 위치가 변하지 않음

sticky : relative 와 fixed 의 조합처럼 작동 / 스크롤 위치에 따라 '붙어있게' 되거나 일반적인 위치에 머무름
