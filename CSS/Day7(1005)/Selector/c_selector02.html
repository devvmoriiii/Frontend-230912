속성 선택자

속성 선택자 : 요소의 속성 및 속성값에 따라 요소를 선택

  • [속성] 선택자 : 해당 속성을 가진 모든 요소를 선택

  • [속성 = 값] 선택자 : 주어진 속성과 속성 값이 정확히 일치하는 요소를 선택

  • [속성 ~= 값] 선택자 : 여러 속성 값 중에 해당 값이 포함되어 있는 요소를 찾아 스타일 적용

  • [속성 |= 값] 선택자 : 속성 값이 정확히 일치하거나 -(하이픈)으로 연결된 단어의 일부인 요소를 선택

  • [속성 ^= 값] 선택자 : 지정한 속성값으로 시작하는 요소
    (Ex. 속성값이 english인 경우 [title ^="eng"]로 적용 가능)

  • [속성 $= 값] 선택자 : 특정 값으로 끝나는 속성을 가진 요소
    (Ex. a[href $= "hwp"], a[href $= "xls"])

  • [속성 *= 값] 선택자 : 값의 일부가 일치하는 속성을 가진 요소
    (Ex. a[href *= "naver"] )
