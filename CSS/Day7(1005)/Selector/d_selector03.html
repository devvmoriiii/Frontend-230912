가상 클래스 & 가상 요소

: 특별한 상황 또는 상태에성 요소를 스타일링

- 가상 클래스 : 요소의 특정 상태를 선택
- 가상 요소 : 실제 요소와는 별개로 페이지의 특정 부분을 선택

사용자의 동작에 반응하는 가상 클래스 선택자 종류

  • 1. :hover : 요소에 마우스를 올렸을 때 선택
    웹 요소에 마우스 커서를 올려놓았을 때 스타일을 적용

  • 2. :active : 요소가 활성화되어 있을 때 선택
    웹 요소를 활성화했을 때의 스타일을 적용

  • 3. :focus : 요소가 포커스되었을 때를 선택
    웹 요소에 초점이 맞추어졌을떄의 스타일을 적용

  • 4. :visited : 요소가 방문한 링크를 선택

  • 5. :link : 방문하지 않은 링크에 스타일을 적용


아래 순서에 따라 가상 클래스가 적용됨.

  1. :link
  2. :visited
  3. :hover
  4. :active
  5. :focus
