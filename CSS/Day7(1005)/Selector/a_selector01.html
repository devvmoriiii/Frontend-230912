CSS 고급 선택자 - 연결 선택자

연결 선택자 (= 컴비네이션 선택자, 조합 선택자) : 선택자와 선택자를 연결해 적용 대상을 제한하는 선택자

  • 1. 동시 선택자: 여러 선택자를 쉼표(,)로 구분하여 한번에 스타일을 적용

  • 2. 하위 선택자: 상위 요소 아래에 있는 '모든' 하위 요소를 선택
    상위 요소와 하위 요소를 나란히 작성 후 띄어쓰기로 구분하여 적용

  • 3. 자식 선택자: 상위 요소 '바로 아래'에 있는 하위 요소만을 선택
    두 요소 사이에 > (부등호) 를 표시해 부모 요소와 자식 요소를 구분 (손자, 손자의 손자 요소에는 적용되지 않음)

  • 4. 인접 형제 선택자: 동일한 부모를 가진 바로 다음의 형제 요소를 선택
    요소1과 요소2 사이에 + 기호를 사용

  • 5. 형제 선택자: 형제 요소들에 모두 스타일 적용
    요소1과 요소2 사이에 ~ 기호를 사용

예시

안녕하세요

오늘은 10월 5일 입니다.

오늘은 목요일 입니다.

만나서 반갑습니다.

제 이름은 이승아입니다.
