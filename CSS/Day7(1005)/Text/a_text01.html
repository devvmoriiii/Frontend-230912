텍스트 스타일

: 사용하는 글자의 모양새를 지정하는 글꼴 스타일 & 웹 문서에서 표시되는 텍스트를 지정하는 문단 스타일

글꼴 스타일

: font- 로 시작하는 속성

  • font-family 속성 : 웹 문서에서 사용할 글꼴을 지정하는 속성
    로컬 폴더 내 지정한 폰트 파일이 없을 경우를 대비하여 2-3개 이상의 글꼴을 , (콤마) 로 구분하여 작성
    글꼴 이름에 띄어쓰기가 필요한 경우 "" 따옴표로 묶어서 표기

  • font-size 속성 : 글자 크기를 지정하는 속성
    글자 크기의 단위는 절대 크기, 상대 크기, 크기, 백분율로 구분할 수 있으며 px (픽셀), pt (포인트), % (백분율) 을 주로 사용

    1. 키워드를 사용하여 글자 크기 지정
      xx-small < x-small < small < medium < large < x-large < xx-large

    2. 단위를 사용하여 글자 크기를 지정 (주로 사용)

  • font-style 속성 : 이탤릭체로 글자를 표시하는 속성
    italic 키워드와 oblique 키워드를 사용 (웹에서는 주로 italic 사용)
    normal 키워드 사용 시 일반적인 형태 (기본값) 으로 표시

  • font-weight 속성 : 글자 굵기를 지정하는 속성

    1. 예약어를 사용 (normal, bold, bolder 등)

    2. 숫자값을 사용 (100 ~ 900 사이)
      400 = normal, 700 = bold

예시

h3 제목입니다.

h4 제목입니다.

문단1 입니다.

문단2 입니다.

문단3 입니다.

문단4 입니다.
