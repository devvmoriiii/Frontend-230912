글꼴 스타일 : "font-" 접두어가 붙는 속성
텍스트 스타일 : "text-" 텍스트의 간격 및 기타 레이아웃 기능에 영향을 주는 속성으로, 글자와 단어/글자로 이루어진 문단에 사용하는 스타일

텍스트 스타일

  • color 속성 : 웹 문서에서 문단이나 제목 등의 텍스트에 글자색을 변경하는 속성
    속성 값은 4가지 방법으로 작성 가능하다.

    1. 16진수 : 0 ~ 9, a ~ f 로 이루어진 16진수이며, "#" 기호 뒤에 6자리 16진수로 작성한다. (ex. #000000 ~ #ffffff)
    2. rgb (| rgba) : red, green, blue 의 값으로 이루어져 있으며, "rgb()" 작성 후 rgb 순서대로 0 (최소) ~ 255 (최대) 의 값을 작성한다. (ex. rgb(0, 0, 255))
      a 는 alpha 의 약자로 투명도를 의미하며, 0 (투명) ~ 1 (불투명) 의 값으로 작성한다. (ex. rgba(0, 0, 256, 0.3))
    3. hsl (| hsla) : hue (색상), saturation (채도), lightness (명도) 의 값으로 이루어져 있으며, "hsl()" 작성 후 h 는 정수, s 와 ㅣ 은 % 단위로 작성한다. (ex. hsl(200, 50%, 30%))
      a 는 alpha 의 약자로 투명도를 의미하며, 0 (투명) ~ 1 (불투명) 의 값으로 작성한다. (ex. hsl(100, 50%, 70%))
    4. 색상 이름 : 색상 영문명을 작성한다. (ex. red, yellow, blue)
