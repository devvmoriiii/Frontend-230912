텍스트 기타 속성

  • text-decoration 속성 : 텍스트의 줄을 표시하거나 제거하는 속성
    하이퍼링크를 적용할 때 기본적으로 생성되는 밑줄을 제거할 때 주로 사용한다.

    1. none : 텍스트 줄 없음
    2. underline : 텍스트 아래에 밑줄 표시
    3. overline : 텍스트 위에 윗줄 표시
    4. line-through : 텍스트 중앙으로 줄 표시 (= 취소선)

  • text-shadow 속성 : 텍스트에 그림자 효과를 추가하는 속성
    사이트 제목처럼 강조해야 할 글자에 주로 사용한다.
    아래 4가지 값을 공백으로 구분하여 작성한다. (ex. 2px 2px 2px #aaa)

      x
    1. 가로 거리 (필수) : 텍스트부터 그림자까지의 거리를 지정한다. (양수 값 : 글자의 오른쪽에 그림자 생성 | 음수 값 : 글자의 왼쪽에 그림자가 생성)
    2. 세로 거리 (필수) : 텍스트부터 그림자까지의 거리를 지정한다. (양수 값 : 글자의 아래쪽에 그림자가 생성 | 음수 값 : 글자의 위쪽에 그림자가 생성)
    3. 번짐 정도 : 그림자의 번짐 정도를 지정하며, 기본값은 0 이다. (양수 값 : 글자의 모든 방향으로 퍼져나가 크게 표시 | 음수 값 : 글자의 모든 방향으로 축소되어 작게 표시)
    4. 색상 : 한 가지 또는 여러 색상이 지정 가능하며, 여러 색상 지정 시 공백으로 구분한다. 기본 값은 현재 글자 색이다.

  • text-transform 속성 : 영문자 표기 시 텍스트의 대소문자를 변경하는 속성 (한글 적용 불가, only 영어 & 기타 외국어 가능)

    1. none: 변환되지 않음 (기본값)
    2. capitalize : 첫 번째 글자를 대문자로 변환
    3. uppercase : 모든 글자를 대문자로 변환
    4. lowercase : 모든 글자를 소문자로 변환

  • letter-spacing 속성 : 글자와 글자 사이의 간격을 조정하는 속성
    word-spacing 속성 : 단어와 단어 사이 간격을 조정하는 속성
