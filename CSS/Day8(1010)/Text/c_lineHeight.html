CSS 단위

  • 숫자 : CSS 에서 사용되는 순수한 숫자 값으로, 일반 수학에서 사용되는 숫자로 값을 표시

  • 길이 : 특정 길이를 나타날 때 사용하며, 여러 단위를 포함하고 있음.

    1. px (픽셀, 포인트) : 절대 단위이며, 화면의 각 픽셀을 기준으로 함.
    2. em : 상대 단위이며, 부모 요소에서 지정한 글꼴의 대문자 M 너비를 기준으로 1em 을 설정함.
    3. rem : 상대 단위이며, 문서 시작 부분(root) 에서 지정한 크기를 기준으로 1rem 을 설정함.

  • 백분율 : 상대 단위이며, 부모 요소를 기준으로 비율을 설정함.

텍스트 높이

  • line height 속성 : 각 텍스트 줄의 높이 (= 줄 간격) 을 조절하는 속성
    권장 줄 높이는 글자 크기의 약 1.5 ~ 2배의 간격이다.
    속성 값은 3가지 방법으로 작성 가능하다.

    1. 길이 지정 (px) : 정수와 CSS 단위를 사용하여 지정 (ex. 글자 크기가 12px 인 p 요소의 줄 간격 >> 18px ~ 24px)
    2. 숫자 지정 : 글자 크기의 배수로 지정 (ex. 1.5, 2)
    3. 백분율 : 배수를 백분율로 지정 (ex. 150%, 200%)

line-height 속성 값과 height 속성 값을 동일하게 지정 시, 텍스트 세로 정렬이 가능하다. (단, 텍스트가 한 줄일 경우에만 적용)
