CSS style sheets

CSS style sheets : 스타일 규칙을 한 눈에 확인하고 필요할 때마다 수정하기 쉽도록 한 군데 묶어 놓은 것

브라우저 기본 스타일 : 웹 브라우저에 기본으로 적용되어 있는 스타일
(ex. h1 태그 사용 시 제목 텍스트의 글자가 크고 긁게 표시 / p 태그 사용 시 제목보다 작게 표시)

사용자 지정 스타일 : 사이트 제작자가 만드는 스타일로, [1. 인라인 스타일 2. 내부 스타일 시트 3. 외부 스타일 시트] 로 분류.

    인라인 스타일 : 스타일 시트를 사용하지 않고, 스타일을 적용할 대상에 직접 표시
  • style 속성이 포함된 한 요소에만 영향을 주는 CSS 선언
  • 유지보수 및 코드의 중복이라는 점에서 사용 X
    내부 스타일 시트 : 웹 문서 안에서 사용할 스타일을 같은 문서 안에서 정리
  • 모든 스타일 정보는 head 태그 안에 정의
  • style 요소 사이에 스타일을 작성
    외부 스타일 시트 : 여러 웹 문서에서 사용할 스타일을 별도 파일로 지정
  • 외부 스타일 시트 확장자는 .css
  • link 태그를 사용하여 외부 스타일 시트 파일을 연결

link 적용 시 폴더 구조에 따른 수신처 작성법

  1. 동일한 폴더 내의 파일 : 파일명.확장자
  2. 동일한 폴더 내의 하위 폴더에 존재하는 파일 : 폴더명/파일명.확장자
  3. 상위 폴더 내의 파일 : ../파일명.확장자
  4. 상위 폴더 내의 다른 폴더 내의 파일 : ../폴더명/파일명.확장자
