CSS Selector (선택자)

선택자 : 웹 문서에서 어느 부분에 스타일을 적용할지 알려주는 기능으로, CSS 규칙의 첫 번째 부분에 해당함

    1. 전체 선택자 (Universal Selector)
  • * (별표) 사용
  • 문서의 모든 항목을 선택 (다른 요소 및 하위 연결자와 함께 연결된 경우 상위 요소 내부)
  • 스타일을 문서의 모든 요소에 적용할 때 사용
  • 주로 웹 브라우저의 기본 스타일을 초기화할 때 사용
    (ex. 웹 문서 내용을 브라우저 창에 바짝 붙지 않도록 존재하는, 문서 내용과 브라우저 테두리 사이에 여백을 제거하거나 또는 ul 태그의 bullet 을 제거할 때 사용)
    2. 타입 선택자 (Type Selector)
  • 타입 선택자 (Type Selector) == 태그 선택자 (Tag Selector) == 요소 선택자 (Element Selector)
  • 타입 선택자를 사용해 스타일을 지정할 경우 해당 태그를 사용한 모든 요소에 적용
    3. 클래스 선택자 (Class Selector)
  • 클래스 이름 앞에 . (마침표) 를 반드시 붙여야 하며, 태그에 class 속성을 사용하여 클래스명을 지정
  • 특정 부분에 스타일을 적용
  • 클래스 이름을 사용하여 다른 선택자와 구별
  • 클래스명은 동일한 문서 내에서 여러 번 지정 가능
  • 하나의 태그에 여러 개의 class 지정 가능
  • 클래스명을 공백 없이 나란히 작성할 경우, 두 개의 클래스명을 모두 가진 요소에만 css 스타일을 적용
    4. 아이디 선택자 (Id Selector)
  • 아이디 이름 앞에 # (샾) 을 반드시 붙여야 함.
  • 특정 부분에 스타일을 한 번만 적용
  • 아이디 이름을 사용하여 다른 선택자와 구별
  • 아이디 선택자는 동일한 문서에서 딱 한 번만 적용 가능
    5. 그룹 선택자 (Group Selector)
  • 여러 선택자에서 같은 스타일 규칙을 사용하는 경우, (쉼표) 로 구분해 여러 선택자를 나열하여 적용 가능
  • 같은 스타일 규칙을 사용하는 요소를 묶어서 사용
    6. 특정 태그 내의 특정 태그에만 속성을 지정하고자 하는 경우
  • 띄어쓰기로 구분하여 선택 가능
  • ex.) footer 태그 내의 p 태그만 선택할 경우 > 'footer p'

hr 태그

hr 태그 : 가로 수평선을 지정하는 태그
