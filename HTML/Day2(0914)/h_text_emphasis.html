중요 (Emphasis)

1. em 태그

em 태그 : Emphasis 의 약자로, 텍스트를 강조 할 때 사용하는 태그

2. i 태그

i 태그 : italic 의 약자로, 텍스트를 기울여 표현할 때 사용하는 태그

# em vs i #

em 태그 : 문장에서 흐름 상 특정 부분을 강조

i 태그 : 단순히 다른 텍스트와 구별하기 위한 의도로 기울여서 표시

강조 (Strong Importance)

1. strong 태그

strong 태그 : 텍스트를 굵게 강조할 때 사용하는 태그

2. b 태그

b 태그 : bold 의 의 약자로, 텍스트를 굵게 표시할 때 사용하는 태그

# strong vs b #

strong 태그 : 경고/주의 사항처럼 중요한 내용을 강조

b 태그 : 단순히 글자만 굵게 표시
