Lists

Unordered List (순서가 없는 리스트)

- 항목의 순서가 중요하지 않을 때 사용
- Unordered List 의 준말로 ul 태그 사용
- 리스트 종류 사용 시, 리스트 항목을 li(= list item) 태그로 감싸서 사용

- 블릿(bullet) : 순서가 없는 목록의 경우 항목 앞에 작은 원이나 사각형을 붙여서 구분

  • Milk
  • Egg
  • Bread
  • Snack

Ordered List (순서가 있는 리스트)

- 각 항목을 순서대로 나열할 때 사용
- Ordered List 의 준말로 ol 태그 사용
- 리스트 종류 사용 시, 리스트 항목을 li(= List Item) 태그로 감싸서 사용

- 순서가 있는 목록은 기본적으로 1, 2, 3 ... 으로 번호 구분

- type 속성을 사용하여 목록의 순서를 나타내는 표현을 변경 가능
- start 속성을 사용하여 시작 번호를 지정 가능

  1. NC vs 삼성 (영대문자)
  2. 롯데 vs 두산 (영소문자)
  3. LG vs KT (로마 숫자 대문자)
  4. KIA vs SSG (로마 숫자 소문자)
  5. 한화 vs 키움 (기본값 숫자)

Definition List (목록 설명)

- 이름(name) 과 값(value) 형태로 된 목록
- 웹 문서에서 목록 형태의 콘텐츠르 삽입할 때 사용

- dl 태그를 이용하여 한 쌍의 dt, dd 태그를 입력
- dt(= Definition Table) 태그를 이용하여 이름(단어명) 부분을 지정
- dd(= Definition Data) 태그를 이용하여 값(설명) 부분 지정
- dt 태그 하나에 여러 개의 dd 태그를 사용 가능
