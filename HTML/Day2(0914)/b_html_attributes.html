HTML 속성

HTML 속성 : 요소에 대한 추가 정보를 제공

- 모든 HTML 요소는 속성을 가질 수 있음
- 모든 속성은 항상 시작 태그에 지정
- 일반적으로 name = "value" 와 같이, 이름 = "값" 형태의 한 쌍으로 구성

HTML 속성 사용 시 주의사항

1. 항상 소문자 속성을 사용
2. 요소 이름 다음에 바로 나오는 속성은 요소 이름과 속성 사이에 공백이 필요하며, 하나 이상의 속성들이 있는 경우 속성들 사이에 공백이 필요함
3. 속성 이름 다음에는 등호(=) 가 붙음
4. 속성 값은 열고 닫는 따옴표로 감싸야 함

HTML 속성 종류

1. href 속성 (hyper reference = 하이퍼 참조)

네이버 스포츠 야구

a 태그 : anchor 의 준말로, 하이퍼링크를 정의

href 속성 : 링크가 다음 페이지로 이동하는 URL을 지정하며, '수신처' 라고 칭함.

2. src 속성 (source = 소스)

src 속성 : img 태그의 필수 속성으로, 표시할 이미지에 대한 경로(URL) 을 지정

2-1. 이미지 경로 지정

1. 절대 경로 : 웹 사이트의 루트 디렉토리에서 시작하여 전체 파일 경로를 지정하는 방식
- 웹 페이지의 위치와 관계없이 항상 같은 리소스를 참조
(Ex. 이미지 주소 복사 >> https://cdn.pixabay.com/photo/2016/01/05/17/51/maltese-1123016_1280.jpg)

2. 상대 경로 : 현재 문서의 위치를 기준으로 하여 파일의 경로를 지정하는 방식
- 상대 경로는 파일의 위치에 따라 지정하는 방식이 상이

  • - 같은 디렉토리의 파일일 경우, filename.jpg
    (Ex. same-directory.jpg)
  • - 하위 디렉토리 파일일 경우, subDirectory/filename.jpg
    (Ex. images/images-folder.jpg)
  • - 상위 디렉토리 파일일 경우, ../filename.jpg
    (Ex. ../diff-directory.jpg)

3. alt 속성 (alter = 바꾸다)

alt 속성 : img 태그의 필수 속성으로, 어떠한 이유로 오류가 있거나 사용자가 스크린 리더를 사용하는 경우에 이미지의 대체 메세지를 지정

4. title 속성

네이버

title 속성 : 링크에 대한 추가 정보를 나타내는 속성으로, a 태그에 마우스 커서를 올릴 시 title 속성의 메세지가 나타남
