HTML 멀티미디어 삽입

object 태그

object 태그 : 오디오, 비디오, PDF 등 다양한 멀티미디어 파일을 삽입하는 태그

- 웹 문서 안에 다른 문서를 삽입할 때도 사용 가능
- widh 속성으로 너비 지정, height 속성으로 높이 지정, data 속성으로 파일 지정

embed 태그

embed 태그 : 오디오, 비디오, 이미지 등 다양한 멀티미디어 파일을 삽입하는 태그

- HTML 초기 버전부터 사용되었으며 대부분의 브라우저에서 사용 가능
- 단일 태그이므로 닫힘 꺽새 필수
- width 속성으로 너비 지정, height 속성으로 높이 지정, src 속성으로 파일 경로 지정

audio 태그

audio 태그 : 오디오, 비디오 파일을 삽입하는 태그

- HTML4까지는 웹 브라우저에서 멀티미디어 파일을 삽입 후 재생 시에 별도의 플러그 인 프로그램(Ex. 곰플레이어 등) 설치가 필수였음
- HTML 에서 배경음악이나 효과음 등의 오디오 파일을 삽입 시 주로 사용

video 태그

video 태그 : 비디오 파일을 삽입하는 태그

audio/video 태그 속성

  • controls 속성 : 플레이어 화면에 오디오/비디오 파일을 시작, 종료할 수 있는 컨트롤 바를 생성
  • autoplay 속성 : 오디오/비디오를 자동으로 실행
  • loop 속성 : 오디오/비디오를 반복 재생
  • mute 속성 : 오디오/비디오의 소리를 제거
  • width, height 속성 : 플레이어의 너비와 높이를 지정하며, 둘 중에 하나만 지정하여도 나머지 속성은 비율을 자동으로 계산
  • poster = "파일 이름" 속성 : 해당 속성이 지정된 비디오가 재생되기 전까지 화면에 표시될 포스터의 이미지를 지정

대부분의 웹 브라우저에서는 오디오나 소리가 있는 비디오 파일의 자동 재생을 금지함
