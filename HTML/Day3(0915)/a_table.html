HTML Table

  • 테이블 (표) : 행(row), 열(column), 셀(cell) 로 이루어짐.
  • 행(row) : 가로
  • 열(column) : 세로
  • 셀(cell) : 표의 내용이 들어가는 한 칸을 의미

table 태그 & caption 태그

table : 표를 만드는 태그로, 표를 생성할 때 사용
caption : 표의 제목 또는 설명을 삽입하는 태그로, 제목이 표의 위쪽 중앙에 표시.
선택적 항목으로 주로 스크린 리더 사용자를 위해 작성

tr, td th 태그

tr 태그 : table row (각 행)

td 태그 : table data (일반적인 테이블 셀)

th 태그 : table header (테이블의 제목 셀을 나타내며, 기본적을 굵은 글씨)

table 태그만으로 표가 생성되지 않으며, 태그 안의 행과 각 행의 셀의 개수를 지정해야 함.

표의 구조를 지정하는 태그

  • thead(제목), tbody(본문), tfoot(요약) 태그 : 표의 구조를 지정할 경우 시각 장애인도 화면 낭독기를 통해 표를 쉽게 이해 가능
  • CSS를 사용하여 각각 다른 스타일 적용 가능
  • 테이블 구조 사용 전후, 눈에 보여지는 테이블은 동일하나 화면 낭독기/컴퓨터 프로그램이 인지하는 구조가 달라짐

테이블 병합

rowspan, colspan 속성을 사용하여 행이나 열을 병합
rowspan : 행 병합
colspan : 열 병합

rowspan 속성 : 특정 셀이 여러 행에 걸쳐 확장되어야 할 때 사용. 'rowspan = "2"' 형태로 사용하며, 해당 셀이 아래쪽으로 2행만큼 확장
colspan 속성 : 특정 셀이 여러 열에 걸쳐 확장되어야 할 때 사용

rowspan 을 사용할 때, 해당 셀 아래에 있는 셀(들)을 생략
colspan 을 사용할 때, 해당 셀 오른쪽에 있는 셀(들) 을 생략
