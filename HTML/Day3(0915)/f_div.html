div 태그

div 태그 : HTML 에서 가장 흔하게 사용되는 구조적 요소 중 하나로, 웹 페이지의 여러 부분을 나누거나 묶는 용도로 사용하는 태그

- 주로 CSS(스타일), JavaScript(기능) 을 적용하기 위해 사용

class, id 속성

- class, id 속성을 사용하여 HTML 태그에 스타일링이나 동작을 지정 가능

class 속성
  • 하나의 HTML 요소에 여러 클래스를 할당 가능 (띄어쓰기로 구분하여 작성)
  • 여러 요소에 동일한 클래스를 할당 가능
  • 클래스 구문 사용 시 '.' (마침표) 문자 뒤에 클래스명을 입력 후 { } (중괄호) 내에서 CSS 속성을 정의
동일한 클래스 사용하기

id 속성
  • 각 HTML 요소는 고유한 id 를 가져야함
  • id 구문 사용 시 '#' (해쉬) 문자 뒤에 아이디명을 입력 후 { } (중괄호) 내에서 CSS 속성을 정의
