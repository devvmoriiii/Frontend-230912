이미지 삽입 태그 (img 태그)

img 태그 : 웹 문서에서 이미지를 삽입할 떄 가장 기본이 되는 태그

- 이미지는 기술적으로 웹 페이지에 삽입되는 것이 아닌, 웹 페이지에 링크되는 개념
- 기본형 :

src(= source) 속성

src 속성 : 이미지 파일의 경로를 지정하여 웹 브라우저에 알려주는 역할

  • 동일한 경로 : 이미지 파일의 이름만 표기(확장자 포함)
  • 하위 폴더 : src 속성에 폴더 이름을 포함하여 표기
  • 다른 서버나 웹 사이트의 이미지 : Full URL 모두 지정

alt(= alter) 속성

alt 속성 : 화면 낭독기 등에서 이미지를 대신하여 읽어 줄 텍스트를 입력

  • 이미지를 제대로 표시할 수 없는 경우에 이미지 대신 텍스트가 나타남

너비와 높이 지정 속성

width 속성 : 이미지의 너비를 지정
height 속성 : 이미지의 높이를 지정

width, height 중 1개만 지정하여도 나머지 속성은 비율을 자동으로 계산

너비와 높이 지정 단위

% (퍼센트) : 이미지 크기의 값을 퍼센트로 지정하며, 현재 웹 브라우저 창(부모 요소)의 너비와 높이를 기준으로 하여 상대적으로 이미지의 크기를 결정


px (픽셀) : 디지털 이미지의 기본 단위로, 절대적인 단위. 한 이미지는 수많은 작은 점들로 구성되어 있으며, 이 작은 점을 픽셀이라고 칭함.
기본 단위이므로 'px' 단위 생략 가능
