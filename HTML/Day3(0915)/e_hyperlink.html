a 태그 & href 속성

a 태그 : 링크를 삽입하는 태그

href 속성 : 링크의 수신처를 입력

- 입력값에 따라 텍스트 링크 또는 이미지 링크로 사용
- 마우스를 링크 위로 이동하면 마우스 화살표가 작은 손으로 변경
- 텍스트레 링크 추가 시 밑줄이 생기고 글자색이 검은색에서 파란색으로 변경
- 방문하지 않은 링크는 파란색 / 1번 이상 클릭한 텍스트 링크는 자주색으로 변경 - 수신처에 'mailto:이메일 주소' 를 입력 시, 해당 메일 주소를 발신으로 하는 메일 쓰기로 연결

하이퍼링크의 target 속성

target 속성 : 링크를 새 탭에서 열어주는 속성으로, 연겨뢴 문서를 열 위치를 지정

  • _self 속성 : 디폴트(기본값) 으로, 링크가 있는 동일한 창/탭에서 문서를 열람 / 롯데 자이언츠_self
  • _blank 속성 : 새 창/탭에서 문서를 열람 / 롯데 자이언츠_blank
    프레임셋을 사용하는 페이지에서 작동하는 속성 값 (HTML5에서는 사용 X)
  • _parent 속성 : 상위 프레임에서 문서를 열람 / 롯데 자이언츠_parent
  • _top 속성 : 창 전체의 본문에서 문서를 열람 / 롯데 자이언츠_top
