input attributes

  • autofocus 속성 : 페이지를 불러오자마자 폼에서 원하는 요소에 자동으로 입력 커서(마우스 포인터) 를 가져다 놓는 속성 (하나의 요소에만 지정 가능)
  • placeholder 속성 : 힌트를 표시해주는 속성으로, 사용자가 텍스트를 입력할 때 입력 란에 적당한 힌트 내용을 표시하고 해당 필드를 클릭 시 내용이 사라짐
  • readonly 속성 : 읽기 전용 필드를 만들어 주는 속성으로, 사용자가 입력 불가하며 읽기만 가능
  • required 속성 : 필수 입력 필드를 지정하는 속성으로, 내용을 폼에 입력한 후 submit 버튼 클릭 시 필수 필드에 필요한 내용이 모두 채여졌는지 검사
  • name 속성 (필수 속성) : 각 입력 필드를 송신하는 속성으로, 해당 속성을 생략할 경우 입력 필드의 값이 전송되지 않음
