input 태그

input 태그 : 사용자로부터 입력을 받을 수 있는 입력 필드를 정의

- 사용자가 입력한 데이터를 서버와 함께 활용할 수 있도록 form 요소 내부에서 사용
- input 요소 자체는 빈 태그(empty tag) 이므로, 속성만을 포함
- label 요소를 사용하여 input 태그의 이름 정의 가능
- 입력 필드의 기본 너비는 20자

input의 기본 속성

type : 입력 필드의 종류나 형태를 정의

name : 입력 필드의 이름을 정의하며, 서버로 데이터를 보낼 때 이 이름을 사용하여 데이터를 식별

value : 입력 필드의 초기 값이나 현재 값을 지정

꼭 알야 할 type 속성 값

type 속성 값을 달리함으로써 input태그의 형태를 여러가지 모양으로 나타낼 수 있음

  • text
  • password
  • email
  • checkbox
  • radio
  • number
  • submit
  • button
  • file
  • hidden
