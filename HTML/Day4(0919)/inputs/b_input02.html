search 필드

search (검색 필드) : 검색을 위한 텍스트 필드

- 모바일 기기의 웹 브라우저에서 해당 필드에 검색을 입력하면 오른쪽에 X 표시가 나타나면서 검색어를 쉽게 지우는 것을 가능하게 함

url 필드

url (주소 필드) : 웹 주소를 입력하는 필드

email 필드

email (이메일 필드) : 이메일 주소를 입력하는 필드

- 입력 값이 지정한 형식에 맞지 않을 경우 웹 브라우저에서 오류 메세지를 보여줌

tel 필드

tel (전화번호 필드) : 전화번호를 나타내는 필드

label 태그의 for 속성 & input 태그의 id 속성

for 속성 : 해당 label 이 설명하고 있는 input 태그를 명시적으로 지정

id 속성 : 해당 input 필드를 고유하게 식별하는 값

- for 속성의 값은 연결하려는 input 태그의 id 속성 값과 일치해야 함.
- label 의 for 속성 값과 일치하는 id 값이 있으면, 해당 label 을 클릭할 경우 자동으로 연결된 input 필드가 활성화
