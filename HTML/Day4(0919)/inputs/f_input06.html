button & file & hidden 필드

button (기본 버튼) : submit/reset 버튼과 같은 기능이 아니며, 버튼의 형태만을 나타냄

- 주로 자바스크립트(기능) 을 실행할 때 사용

file (파일 첨부) : 사진이나 문서를 첨부하는 필드로, 웹 브라우저 화면에 [파일 선택] or [찾아보기] 버튼이 표시

- 첨부한 파일을 서버로 전송하는 기능은 추가적으로 JS 등의 웹 프로그래밍 언어를 사용해야 함
- 초기 값은 '선택된 파일 없음'

hidden (숨겨진 필드) : 화면의 폼에 보여지지 않으며, 사용자가 입력을 마치면 폼과 함께 서버로 전송되는 요소

- 사용자에게 보여지지 않으면서 관리자가 알아야 하는 정보를 입력
