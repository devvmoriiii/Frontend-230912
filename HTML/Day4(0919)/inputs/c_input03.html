checkbox & radio 필드

checkbox & radio : 여러 항목 중에서 원하는 항목을 선택할 때 사용하는 폼 요소

checkbox : 2개 이상 선택 가능

radio : 1개만 선택 가능하며, 이미 선택한 항목이 있을 경우 다른 항목을 선택하면 기존 항목이 취소

checkbox & radio 속성

value 속성 (필수 속성) : 선택한 체크 박스나 라디오 버튼을 서버에 알려줄 때 넘겨줄 값을 지정
영문 or 숫자로 작성

checked 속성 : 여러 항모 중에서 기본으로 선택해 놓고 싶은 항목을 지정
속성 값 없이 키워드 작성만으로도 사용 가능

name 속성 (radio 사용 시 필수 속성) : 라디오 버튼/체크 박스가 여러 개 있는 경우, 서버 폼 프로그램에서 구분을 하기 위해 이름을 지정
radio 버튼은 여러 개 중에서 한 가지만 선택 가능하므로, 관련 있는 그룹끼리는 동일한 name 속성 값을 가져야 함
(지정하지 않을 경우 서로 다른 radio 투표로 인식하여 중복 선택이 가능해짐)

예시

2023 프로야구 포스트시즌 진출팀을 예측해주세요.

2023 프로야구 포스트시즌 진출팀 예측 이벤트 결과 문자 수신 동의
