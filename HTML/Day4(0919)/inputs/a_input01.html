text & password 필드

text (텍스트 필드) : 주로 아이디나 이름, 주소 등 한 줄 짜리 일반 텍스트를 입력할 때 사용

password (비밀번호 필드) : 입력되는 내용을 화면에 보여주지 않아야 하므로 * (별표) or ● (불릿) 으로 표시


text, password 필드 주요 속성

size 속성 : 화면에 몇 글자가 보이도록 할 것인지 지정

Ex.

value 속성 : 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여주는 내용
해당 속성을 사용하지 않으면 빈 텍스트 필드가 표시되며, password 필드에서는 사용하지 않음 주로 회원정보 수정 시 기본 정보를 가져와서 텍스트 필드에 표시

Ex.

maxlength 속성 : 필드에 입력할 수 있는 최대 문자 수를 지정
minlength 속성 : 필드에 입력해야 하는 최소 문자 수를 지정

Ex.
