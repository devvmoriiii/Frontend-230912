HTML Element (요소)

HTML Element (요소) : 시작 태그 + 일부 내용 + 끝 태그로 구성되어 있으며, 시작 태그에서 끝 태그까지의 모든 것

끝 태그는 필수로 입력되어야 하며, 끝 태그 미입력 시 프로그램 오류 발생 가능

HTML 은 요소 중첩이 가능

요소 중첩 : 요소가 다른 요소를 포함 가능

모든 HTML 문서는 중첩된 HTML 요소로 구성

요소의 태그 순서를 중첩해서는 X

빈 HTML 요소

빈 HTML 요소 : '단일 태그' 라고도 하며 br 태그와 같이 내용이 없는 HTML 요소. 빈 HTML 요소는 종료 태그가 없음

br 요소

br 요소 : 줄바꿈을 하는 요소

img 요소

img 요소 : 해당 위치에 이미지를 삽입하기 위한 요소


대소문자 구분

HTML은 대소문자를 구분하지 않음.

단, XHTML 은 엄격한 문서 유형을 요구하므로 소문자로만 작성해야 함. 단일 태그의 닫힘 표시(/) 도 필수

p태그는 문단을 구분하는 태그입니다.

태그 vs 요소

태그 : 태그 자체를 가르킴.

요소 : 태그를 적용한 것을 가르킴.
