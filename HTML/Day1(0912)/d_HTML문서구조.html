(1-1) HTML 정의 HTML : HyperText Markup Language 의 약자로, 웹 문서를 만드는 언어 (프로그래밍 언어 X, 마크업 언어 O) HyperText : 문서를 서로 '연결' 해주는 링크 Markup : '태그' 를 이용하여 웹 브라우저에 내용의 위치를 '표시' (1-2) 일반 문서 vs 웹 문서 일반 문서 : 내용 입력 프로그램과 입력 내용 확인 프로그램이 동일 (Ex. 워드, 엑셀, 메모장) 웹 문서 : 내용 입력 프로그램과 입력 내용 확인 프로그램이 다름 (!=) 웹 편집기(web editor) 에서 마크업(Markup) 형식으로 작성 >> 웹 브라우저(web browser) 에서 인식하여 표시 (1-3) 태그 태그 : 웹 브라우저가 각 구분을 구별하여 인식하기 위해 붙이는 꼬리표 '<' , '>' 꺽새 (부등호) 표시를 이용하여 작성 (2) HTML 웹 문서의 확장명 .html : HTML 파일의 확장명 (2-1) 에밋(Emmet) 기능 Emmet 기능 : HTML, CSS 의 작성 시간을 단축시켜주는 에디터의 확장 기능으로, 일종의 코드 라이브러리 개념 미리 정해진 문자로 태그를 조합해, 약어 문자열을 입력 시 약어를 원래의 코드로 대체하여 확장해주는 기능 ! 에밋 : HTML 문서 구조를 작성해주는 Emmet 기능 (2-2) HTML 주석 주석 : 브라우저에 표시되지 않으며, HTML 소스 코드를 문서화하는 것을 도움. 즉, 코드에 설명을 덧붙이는 경우에 사용 ' < ! - - - - > ' 와 같이 꺽새와 느낌표, 하이픈 4개로 구성 (3) HTML 기본 구조 HTML 문서 구조
