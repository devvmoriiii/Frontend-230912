HTML Form 태그 종류

input 태그 : 폼에서 한 줄 정도로 간단한 내용을 입력 받는 태그

textarea 태그 : 여러 줄을 입력하는 텍스트 영역 태그 (Ex. 게시판 글 입력, 회원 가입 양식에서 사용자 약관)

select & option 태그 : 드롭다운 목록을 만들어주는 태그로, 사용자가 내용을 직접 입력하지 않고 '여러 옵션 중에서 선택' 가능
select 태그 : 드롭다운 목록을 생성하는 태그로, 목록 클릭 시 옵션이 요소 아래쪽으로 펼쳐짐. 목록의 시작과 끝을 표시
option 태그 : 데이터 목록을 생성하는 태그로, 원하는 항목을 추가

datalist & option 태그 : 데이터 목록을 만들어주는 태그로, 텍스트 필드에 값을 직접 입력하지 않고 '미리 만들어 놓은 값' 중에서 선택
(input 태그에 사전에 정의된 옵션을 제공하기 위해 사용 및 자동 완성 기능을 사용하기 위해 사용)
datalist 태그 : 데이터 목록의 시작과 끝을 표시
option 태그 : 각 데이터의 옵션을 표시

button 태그 : 버튼을 만들어주는 태그로, 폼을 전송하거나 리셋하는 버튼을 삽입 가능 (!= input 태그의 type="button" 형태는 오직 버튼의 형태만 삽입)
화면 낭독기로 웹 문서를 읽을 때 button태그를 만나면 이 부분에 버튼이 있다는 것을 명확하게 전달
버튼을 클릭해서 자바스크립트(기능) 을 실행할 때 사용

HTML Form 태그 속성

    textarea 태그
  • name 속성 : 폼 제출 시 데이터를 구별하기 위한 이름
  • rows 속성 : 텍스트 영역의 세로 길이를 줄 단위로 지정하며, 지정한 숫자보다 줄 개수가 많아질 경우 자동으로 스크롤 막대 생성
  • cols 속성 : 텍스트 영역의 가로 너비를 문자 단위로 지정하며, 지정하는 글자 수는 영문자를 기준 (한글 1글자 == 영문 2글자)
  • placeholder 속성 : 사용자에게 입력 메시지나 설명을 보여주기 위한 텍스트
    단, 열림 태그와 닫힘 태그 사이에 문자열 요소가 있을 경우, 해당 문자열 요소가 우선 나타나고 그 문자열을 지우면 placeholder 가 나타남
    select 태그
  • name 속성 : 폼 제출 시 데이터를 구별하기 위한 이름
  • size 속성 : 화면에 표시할 드롭다운 항목의 개수를 지정. 기본값은 1 이며 맨 처음 Option 항목 한 개와 함께 화살표가 나타남.
  • multiple 속성 : 여러 옵션을 선택할 수 있도록 지정 (ctrl 키를 누른 후 다중 선택 가능)
    option(select) 태그
  • value 속성 : 해당 항목을 선택할 때 서버로 넘겨줄 값을 지정
  • selected 속성 : 해당 옵션을 기본으로 선택
    datalist 태그
  • id 속성 : input 의 list 속성과 연결하기 위한 식별자
    option(datalist) 태그
  • value 속성 : 서버로 넘겨주는 값을 지정
    button 태그
  • submit : input 태그 type 의 submit과 동일
  • reset : input 태그 type 의 reset과 동일
  • button : input 태그 type 의 button과 동일
